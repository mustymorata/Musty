Sports Performance Metrics Tracker

Sports Analytics Excel Datasets HTML Extractors

An academic analysis prototype designed to aggregate, clean, and visualize football player performance figures collected from various Nigerian national soccer portals.

Project Overview & Engineering Pipeline
1. Context and Problem Statement

Local sports analytics suffer from a major accessibility issue. Detailed performance variables like Expected Goals (xG), match action distributions, and passing accuracy metrics are rarely packaged in unified formats for local Kano squads.

2. Data Acquisition Method

Using semantic structures, I mapped local club websites, isolating team and player tables. These HTML blocks were programmatically extracted and parsed, cleaning out noisy visual code to keep the numerical variables.

3. Processing & Analytics Insights

The output was structured into spreadsheets. Utilizing pivot tables and logic arrays, I created ranking models that categorize forwards by shot-conversion efficiency, supporting local tactical scouts with clean data insights.

Key Analytical Takeaways
  • Total Scraped Matches: 42 Fixtures
  • Variables Extracted: Assists, Goals, Pass Completion %, Key Tackles
  • Tooling: Excel Macros, Semantic HTML Extraction Protocols
Project Impact

Successfully constructed an automated dashboard structure demonstrating my capacity to turn unorganized sports records into reliable performance intelligence.
